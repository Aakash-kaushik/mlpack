Implementation of the Radial Basis Function layer. 

The \doxyref{RBF}{p.}{classmlpack_1_1ann_1_1RBF} class when use with a non-\/linear activation function acts as a Radial Basis Function which can be used with Feed-\/\+Forward neural network.
