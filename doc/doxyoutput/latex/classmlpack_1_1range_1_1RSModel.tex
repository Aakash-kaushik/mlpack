\doxysubsection{Detailed Description}
The \doxyref{RSModel}{p.}{classmlpack_1_1range_1_1RSModel} class provides an abstraction for the \doxyref{Range\+Search}{p.}{classmlpack_1_1range_1_1RangeSearch} class, abstracting away the Tree\+Type parameter and allowing it to be specified at runtime. 

This class is written for the sake of the {\ttfamily range\+\_\+search} binding, but is not necessarily restricted to that usage.
