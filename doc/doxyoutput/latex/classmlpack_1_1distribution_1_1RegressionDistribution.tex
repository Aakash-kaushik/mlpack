\doxysubsection{Detailed Description}
A class that represents a univariate conditionally Gaussian distribution. 

Can be used as an emission distribution with the hmm class to implement HMM regression (HMMR) as described in {\texttt{ https\+://www.\+ima.\+umn.\+edu/preprints/\+January1994/1195.\+pdf}} The hmm observations should have the dependent variable in the first row, with the independent variables in the other rows.
