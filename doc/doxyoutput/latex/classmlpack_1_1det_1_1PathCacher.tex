\doxysubsection{Detailed Description}
This class is responsible for caching the path to each node of the tree. 

Its instance is provided to \doxyref{Enumerate\+Tree()}{p.}{namespacemlpack_1_1tree_abab37b0038f14ab4dcdf83c60468e923} utility ONCE and it caches the paths to all the leafs and then easily (and quickly) retrieves these paths for each test entry.
