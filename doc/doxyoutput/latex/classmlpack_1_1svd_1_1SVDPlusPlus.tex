\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Optimizer\+Type = ens\+::\+Standard\+SGD$>$\newline
class mlpack\+::svd\+::\+SVDPlus\+Plus$<$ Optimizer\+Type $>$}

SVD++ is a matrix decomposition tenique used in collaborative filtering. 

SVD++ is similar to \doxyref{Bias\+SVD}{p.}{classmlpack_1_1svd_1_1BiasSVD}, but it is a more expressive model because SVD++ also models implicit feedback. SVD++ outputs user/item latent vectors, user/item bias, and item vectors with regard to implicit feedback. Parameters are optmized by Stochastic Gradient Desent(\+SGD). The updates also penalize the learning of large feature values by means of regularization.
