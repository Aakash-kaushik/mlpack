class mlpack\+::math\+::\+Range\+Type$<$ T $>$}

Simple real-\/valued range. 

It contains an upper and lower bound.

Note that until mlpack 3.\+0.\+0, this class is named Range\+Type$<$$>$ and for the specification where T is double, you can use \doxyref{math\+::\+Range}{p.}{namespacemlpack_1_1math_ad17d9cabd4fd82cbed4ccd5e53b47d70}.
