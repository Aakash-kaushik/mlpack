class mlpack\+::kmeans\+::\+Naive\+KMeans$<$ Metric\+Type, Mat\+Type $>$}

This is an implementation of a single iteration of Lloyd\textquotesingle{}s algorithm for k-\/means. 

If your intention is to run the full k-\/means algorithm, you are looking for the \doxyref{mlpack\+::kmeans\+::\+KMeans}{p.}{classmlpack_1_1kmeans_1_1KMeans} class instead of this one. This class is used by \doxyref{KMeans}{p.}{classmlpack_1_1kmeans_1_1KMeans} as the actual implementation of the Lloyd iteration.
